# Deadwater Gulch: BOOMTOWN (frontier v1 — the reference build)

Event-driven Western town-boss game. You founded a gold town; the gold WILL run out
(the HUD bar is the clock). Convert the boom into a railroad, a cattle pivot, or rob
your own bank and skip town — before the bust.

Built by the frontier model as the measuring stick for cheaper-model builds, from
PLAN-boomtown.md. TypeScript strict + three.js 0.184 WebGPU + TSL node materials.

Play: tap **End Week**, answer the event cards, read the Gazette. Drag = orbit,
pinch/wheel = zoom. Needs a WebGPU browser (Android Chrome works).

`npm i && npm run dev` · `npm test` (34 specs) · `npm run build` (tsc gate included)
